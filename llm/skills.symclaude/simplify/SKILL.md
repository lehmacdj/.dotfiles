Use the Task tool with `subagent_type: "code-simplifier:code-simplifier"` to simplify recent changes in the codebase.

The code-simplifier agent focuses on:
- Improving clarity and readability
- Ensuring consistency with existing patterns
- Enhancing maintainability
- Preserving all functionality

Pass any arguments provided by the user to help focus the simplification (e.g., specific files or areas to focus on).
